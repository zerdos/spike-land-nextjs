## Executive Summary

This epic transforms Spike Land from a "Vibe Coded Apps with Claude Code" platform into an "AI-Powered Social Media Command Center" with **Orbit** as the core product. The platform pivots from targeting developers to targeting social media managers, content creators, and small business marketing teams.

### Key Strategic Changes

- **Old Positioning**: Developer tool for building apps with AI agents
- **New Positioning**: Professional social media management platform
- **Core Product**: Orbit (Social Media Command Center)
- **Secondary Products**: Pixel (image enhancement), Vibe Coding (campaign tools)
- **Monetization Shift**: From token-based to subscription-based ($0 / $29 / $99)

## Phase 6: Analytics Dashboard (#842)

### Status: PENDING

### Objective

Create AI-powered analytics dashboard with engagement insights, growth tracking, and AI-generated recommendations.

### Implementation Steps

#### 1. Database Schema Updates

**File**: `prisma/schema.prisma`

Add new model for AI insights:

\`\`\`prisma
enum InsightType {
OPPORTUNITY
WARNING
ACHIEVEMENT
TREND
}

model AIInsight {
id String @id @default(cuid())
workspaceId String
workspace Workspace @relation(fields: [workspaceId], references: [id], onDelete: Cascade)
type InsightType
title String
description String @db.Text
recommendation String? @db.Text
metrics Json
confidence Float
isRead Boolean @default(false)
createdAt DateTime @default(now())
updatedAt DateTime @updatedAt

@@index([workspaceId, isRead])
@@index([workspaceId, createdAt])
}
\`\`\`

\`\`\`bash
