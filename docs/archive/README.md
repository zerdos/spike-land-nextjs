# Documentation Archive

This directory contains historical and archived documentation for the Spike Land platform.

## Purpose

Documents in this archive are:

- **Historical**: Previous versions of documentation that have been superseded
- **Deprecated**: Documentation for features that are no longer active
- **Reference**: Old implementation details that may be useful for understanding system evolution

## Organization

Archive files are organized chronologically and by topic. When archiving documentation:

1. Include the date in the filename (e.g., `FEATURE_NAME_2024-12-01.md`)
2. Add a header explaining why the document was archived
3. Link to the current version of documentation if applicable

## Active Documentation

For current documentation, see the parent `docs/` directory or refer to [CLAUDE.md](../CLAUDE.md) for a complete documentation index.
